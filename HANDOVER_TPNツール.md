## tpn_ppn_planner.html の機能

### タブ構成

| タブ | 内容 |
|---|---|
| 📊 栄養目標 | カロリー・タンパク目標、NPC/N比、GIR |
| 🧪 推奨組成 | AA液/ブドウ糖/脂肪乳剤の推奨量・推奨プランA/B・側管設定中の水分案内 |
| 💊 製剤選択 | 採用製剤一覧・mL入力・電解質インライン表示・水分バランス（側管含む） |
| 📋 投与計画 | リフィーディング対応Step-upプロトコル |
| ⚗️ 電解質 | Na/K/Ca/Mg/P/Zn のバランス評価（側管分含む） |
| 📄 輸液指示 | 全輸液（TPN＋側管）のまとめ・1日合計（カロリー/水分/Na/K/タンパク/脂肪） |

### 左パネル入力項目
- 患者情報：身長・体重・年齢
- 臨床設定：ICU急性期/ICU回復期/一般病棟（標準/高栄養）
- ストレス係数・タンパクレベル・投与ルート・期間
- 目標水分量・経口カロリー・脂質比率
- **💧 側管・追加輸液**（間欠投与 or 持続側管）

## 計算仕様

### カロリー目標

| 臨床設定 | kcal/kg/日 |
|---|---|
| ICU急性期（Day 1–3） | 20 |
| ICU回復期（Day 4+） | 25 |
| 一般病棟（標準） | 27.5 |
| 一般病棟（高栄養要求） | 32.5 |

### タンパク目標

| 設定 | g/kg/日 |
|---|---|
| 低（維持） | 0.8–1.0 |
| 中（中等度ストレス） | 1.2–1.5 |
| 高（重症・熱傷） | 1.5–2.0 |

### 補正体重
BMI ≥ 30 の場合: `IBW + 0.25 × (実体重 - IBW)`（IBW = 身長m² × 22）

### リフィーディング高リスク判定
`BMI < 16` または `ICU設定 かつ BMI < 18.5`

- 高リスク: Day1-2 → 10 kcal/kg、Day3 → 15、Day4 → 20、Day5 → 25、Day6+ → 目標
- 通常リスク: Day1 → 50%目標、Day2 → 75%、Day3+ → 目標

### 側管投与量計算
- 間欠投与: `1日量(mL) = mL/本 × 本数/日`
- 持続側管: `1日量(mL) = mL/h × 24`
- 電解質: `投与量(mEq) = Na(mEq/L) × 1日量(mL) / 1000`

## 変更履歴

### 2026-05-10（このセッションでの変更）

1. **推奨組成タブから「個別混合参考」を削除**
   - TPN規格品を使うのが通常のため不要と判断
   - `renderComposition()` 内の `aaVol`/`glcVol`/`fatVol`/`baseVol`/`addVol` 変数と対応テーブル・警告アラートを削除

2. **推奨組成タブに側管水分の案内を追加**
   - 側管が設定されている場合のみ「側管 X mL 設定済み。TPN/PPN製剤でさらに X mL を補う想定」のinfoアラートを表示
   - `buildRecommendSection()` の直前にIIFEで挿入

3. **製剤選択タブの水分バランス表示を強化**
   - 側管設定時：「💧 水分バランス（側管含む）」ブロックを表示（TPN製剤 / 側管・追加輸液 / 合計 の3行）
   - 合計が目標超過の場合は赤字＋⚠️表示
   - 側管なし時は従来の「残り X mL 補充」アラートをそのまま表示

4. **側管変更時の連動を修正**
   - `addDrip()` / `removeDrip()` / `updateDrip()` / `updateDripCalc()` の4関数に `renderFormulationResult()` と `renderComposition()` の呼び出しを追加
   - 側管の数値入力中もフォーカスを維持しつつ、右パネルの水分バランス・推奨組成がリアルタイム更新される

5. **輸液指示タブの1日合計にタンパク・脂肪を追加**
   - `renderOrderSheet()` の製剤集計ループに `tpnAA`・`tpnFat` を追加
   - 1日合計テーブルに「タンパク（AA）」「脂肪」の2行を追加
   - タンパクは目標±10〜20%以内で✅、不足時は「⚠️ 不足 X g」表示
   - 脂肪は目標との比較を表示（脂肪乳剤なし時は「脂肪乳剤なし」）
