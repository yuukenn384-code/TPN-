# 栄養管理ツール（TPN-）

成人・医師個人使用の栄養管理ツール集。**このリポジトリが正本**で、GitHub Pages で配信される版が唯一の最新版です。

公開URL（iPhoneホーム画面追加可・PWA）：https://yuukenn384-code.github.io/TPN-/

## 収録
| ファイル | 内容 |

データは端末内のみ（外部送信なし）。実際の臨床判断は医師の責任で行う。

## 改訂のしかた（単一正本ワークフロー）
編集は**このローカルクローン（`~/TPN-`）でのみ**行う。手元コピーは作らない（版管理は git 履歴に一本化）。

```bash
cd ~/TPN-
git pull                       # 最新化
# enteral.html / tpn.html / index.html を編集
#   （ブラウザでローカルプレビュー・iPhone実機相当を確認）
git add -A && git commit -m "改訂内容"
git push                       # GitHub Pages に自動反映
```

- 過去版に戻したいときは git 履歴から（`git log` → 任意コミットへ）。手動バックアップは不要。
- iCloud `経管栄養・TPNツール/` は旧素材を退避済みで**廃止**（編集対象ではない）。
- 改訂は「設計合意」「push前の実機＋医学的妥当性チェック」の関所を通す。
